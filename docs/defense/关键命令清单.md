# 关键命令清单

## 1. 启动集群

```powershell
minikube start -p minikube2 --driver=docker
kubectl config use-context minikube2
kubectl get nodes
```

## 2. 构建并加载镜像

```powershell
docker build -t thesis-sample-api:latest services/sample-api
minikube -p minikube2 image load thesis-sample-api:latest
```

## 3. 部署基础服务

```powershell
kubectl apply -f deploy/base/namespace.yaml
kubectl apply -f deploy/base/deployment.yaml
kubectl apply -f deploy/base/service.yaml
kubectl -n thesis-demo rollout status deployment/sample-api --timeout=300s
```

## 4. 检查资源指标

```powershell
minikube -p minikube2 addons enable metrics-server
kubectl top nodes
kubectl top pods -n thesis-demo
```

## 5. 部署监控和自定义指标

```powershell
kubectl apply -f deploy/monitoring/prometheus.yaml
kubectl apply -f deploy/monitoring/prometheus-adapter.yaml
kubectl get --raw "/apis/custom.metrics.k8s.io/v1beta1"
```

## 6. 应用伸缩策略

```powershell
kubectl apply -f deploy/hpa/hpa-cpu.yaml
kubectl apply -f deploy/hpa/hpa-multi.yaml
kubectl apply -f deploy/vpa/vpa-initial.yaml
kubectl -n thesis-demo get hpa
kubectl -n thesis-demo get vpa
```

## 7. 运行正式实验

```powershell
powershell -ExecutionPolicy Bypass -File scripts/run_experiments.ps1
```

## 8. 单独汇总和出图

```powershell
python results/aggregate_experiment.py --config scripts/experiment.config.json --raw-root results/raw --out results/processed/experiment_metrics.csv
python results/plot_results.py --input results/processed/experiment_metrics.csv --outdir results/figures
```

## 9. 快速查看结果

```powershell
Import-Csv results/processed/experiment_metrics.csv | Group-Object strategy | Select-Object Name,Count
Import-Csv results/processed/experiment_metrics_summary.csv | Format-Table -AutoSize
Get-ChildItem results/figures -Filter *.png
```
